下标
下标使用包裹，例如：H2~O将产生水的分子式。H2O

上标
上标使用包裹，例如：y2=4将产生表达式 : y^2 = 4

插入表情

使用:happy:输入高兴的表情 :happy: ，使用:sad:输入悲伤的表情 😓 ，使用😢 输入哭的表情😢 等。以此类推！

下划线

用HTML的语法Underline将产生下划线Underline. 我有一条下划线

删除线
使用~~包裹的文本将会具有删除的样式，例如删除文本将产生删除文本的样式。

代码
使用` 包裹的内容将会以代码的样式显示，例如

printf()

输入~~~ 或者 ``` 回车，可以输入代码块，并且可以选择代码的语言。例如：

public class HelloWorld{
Console.WriteLine("Hello World!");
}
强调 斜体
使用两个*号或者两个_包裹的内容将会被强调。例如

使用两个*号强调内容 使用一个* 倾斜
使用两个下划线强调内容 使用一个_ 倾斜
我被强调了 倾斜

我被强调了 倾斜

插入图片
我们可以通过拖拉的方式，将本地文件夹中的图片或者网络上的图片插入。

测试图片.png
插入URL连接
使用尖括号包裹的url将产生一个连接，例如：<www.baidu.com>将产生连接:<www.baidu.com>.

如果是标准的url，则会自动产生连接，例如:www.google.com

目录列表Table of Contents（TOC）

输入[toc]然后回车，将会产生一个目录，这个目录抽取了文章的所有标题，自动更新内容。

水平分割线
使用***或者---，然后回车，来产生水平分割线。

标注
我们可以对某一个词语进行标注。例如

某些人用过了才知道[1]

某些人用过了才知道[1]

把鼠标放在注释上，将会有提示内容。
